Overview

This was one of the projects that the SAAS Web Development committee undertook in Spring 2020, and is still ongoing. We are creating an e-commerce web app to sell club merch (shirts, stickers). The app allows user authentication — it includes admin pages that dislay inventory and all user orders, as well as user pages that display the user’s past orders. In addition to coding, I took on the task of designing the UI for the web app.

Design Justification

I wanted the design of the web app to be consistent with the design of the SAAS website. This consistency includes the gradient under the top navigation bar, white boxes with shadows for each merch item, and a large gradient wave in the background. Overall, I wanted to convey a friendly and playful vibe through the gradient wave with soft curve, icons/graphics, and pastel colors.

Merch Page

When the web app is launched, the user is first taken to a page that shows all the merch that they can purchase. The user can filter by category, choosing to see shirts, stickers, or both.

Checkout Page

When the user clicks on the cart icon in the navigation bar, they are taken to the checkout page. They are given a chance to review and edit their order in the “Your Shopping Cart” box before entering their payment information in the “Confirm Order” box.

After submitting their order and payment information, the app displays the user’s order summary and a thank you message.
